This project configures an ESP32-H2 into a battery-friendly Zigbee End Device that monitors ambient conditions using dual DS18B20 temperature sensors connected to a shared bus. By default, the firmware automatically initializes the 1-Wire protocol on GPIO 4 using the ESP-IDF RMT peripheral backend, discovering and mapping two independent sensor pointers. The device executes parallel hardware conversions and unconditionally transmits Zigbee cluster reports for both Endpoint 1 and Endpoint 2 every 5 minutes.
