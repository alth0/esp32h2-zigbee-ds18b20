This project configures an ESP32-H2 into a battery-friendly Zigbee End Device that monitors ambient conditions using a DS18B20 temperature sensor. By default, the firmware automatically initializes the 1-Wire protocol on GPIO 4 using the ESP-IDF RMT peripheral backend. It instantly transmits a Zigbee cluster report if the temperature shifts by $\pm 0.5^\circ\text{C}$, or forces a fallback heartbeat transmission every 30 minutes to stay active.
